Projects

  1. Progressive Knowledge Distillation for Graph Neural Architecture Search in Distributed Environments - The project is about creating a novel knowledge distillation framework for graph neural architecture search in distributed environments. It aims to improve model selection and training processes and implement progressive knowledge distillation techniques..
  2. Ranking Prediction-based Spectral Graph Neural Architecture Search Framework for Graph Anomaly Detection The project aimed at enhancing graph anomaly detection. I designed a novel spectral graph neural architecture search framework using ranking prediction methods to improve model selection and detection accuracy. Our research is currently under review at the Neural Networks Journal, and the project's code is available on GitHub here.
  3. Cancer drug response prediction with surrogate modeling-based graph neural architecture search - The project focusing on cancer drug response prediction. The work involved data preparation, surrogate modeling, graph neural networks, and extensive architecture search, resulting in a published paper in Bioinformatics. You can access the project's code on GitHub here..
  4. Prediction-based graph neural architecture search - We developed a novel framework, GraphNeural Architecture Prediction (GraphNAP), which included a neural predictor to efficiently select optimal Graph Neural Network (GNN) architectures. Our work significantly advanced automatic graph neural architecture search, with research papers published in Machine Learning and Knowledge and Information Systems journals. You can find the project's code on GitHub here..
